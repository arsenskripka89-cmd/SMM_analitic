# SMM Analytics

Мінімалістичний інтерфейс для перегляду SMM-показників у браузері.

## Швидкий запуск у браузері
1. Перейдіть до фронтенду та встановіть залежності:
   ```bash
   cd frontend
   npm install
   ```
2. Запустіть дев-сервер:
   ```bash
   npm run dev
   ```
3. Відкрийте у браузері `http://localhost:3000` і перейдіть на потрібну сторінку через верхнє меню.

Усі ключові розділи (Dashboard, Analytics, Accounts, Settings, Auth) доступні одразу після старту без додаткової конфігурації.
